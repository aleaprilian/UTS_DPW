
# Re-Design Website FT UNRI

https://ft.unri.ac.id/id merupakan website resmi FT UNRI yang juga sebagai pusat informasi publik FT UNRI.Pada repository ini, akan dilakukan re-design website FT UNRI untuk memenuhi tugas UTS Desain Pemrograman Web.Beberapa perubahan design yang kami lakukan diantaranya : 
- Redesign Body Website, dengan penyesuaian isi konten untuk meningkatkan user experience pengguna
- Redesign Footer Website, dengan penyesuaiian penggunaan warna agar nuansa footer sesuai dengan konten website 
- Redesign tampilan pada mobile dengan penyesuaian offcanvas agar tidak menganggu navigasi pengguna
- Menambahkan section tentang pengunguman, berita dan prestasi mahasiswa sehingga memudahkan dalam penyebaran informasi
